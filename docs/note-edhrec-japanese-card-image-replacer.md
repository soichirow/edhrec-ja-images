# EDHRECのカード画像を日本語版に自動で差し替えるTampermonkeyスクリプトを作りました

EDHRECを見ながら統率者デッキを組んでいるとき、カード画像やカード名が英語のままだと、あとで日本語名を調べ直すのが地味に面倒です。

そこで、EDHREC上のカード画像をScryfallにある日本語印刷版の画像へ自動で差し替えるユーザースクリプトを作りました。

Chrome拡張として配布するのではなく、Tampermonkeyに入れて使う形です。ブラウザにTampermonkeyを入れて、Greasy Forkからスクリプトをインストールするだけで使えます。

## できること

- EDHRECのカード画像を日本語印刷版の画像へ自動で差し替える
- 日本語印刷版がないカードはScryfallの英語通常版へ差し替える
- 日本語画像が見つかるまでは、EDHRECの元の英語画像をそのまま表示する
- スクロール前にカード情報と画像を先読みして、表示を少し速くする
- 横長サムネイルなどカード形ではない表示は、潰れ防止のため差し替えない
- 差し替えたカード名と操作ボタンを画像の下に表示する
- 日本語名をクリックするとScryfallを開ける
- `コピー` ボタンでカード名をコピーできる
- `★` ボタンでお気に入り登録できる
- お気に入り一覧をまとめてコピーして、Excelやスプレッドシートに貼れる
- 晴れる屋、BIG MAGIC、シングルスター、東京MTG、メルカリの検索リンクを開ける
- 読み仮名つきの日本語名は、コピーしやすい表記へ整形する

たとえばScryfall側で次のような表記が返ってきても、

```text
ファイレクシアの変（へん）形（けい）者（しゃ）
```

画面表示とコピーでは次の形になります。

```text
ファイレクシアの変形者
```

## インストール方法

一番簡単なのはGreasy Forkから入れる方法です。

1. Tampermonkeyを入れる  
   https://www.tampermonkey.net/
2. スクリプトページを開く  
   https://greasyfork.org/ja/scripts/580860-edhrec-japanese-card-image-replacer
3. `Install this script` を押す
4. Tampermonkeyの確認画面で `インストール` を押す
5. EDHRECを開く  
   https://edhrec.com/

これでEDHREC上のカード画像が順番に差し替わります。

ソースコードはこちらです。

```text
https://github.com/soichirow/edhrec-ja-images
```

GitHub Rawから直接インストールする場合はこちらです。

```text
https://raw.githubusercontent.com/soichirow/edhrec-ja-images/main/userscript/edhrec-ja-images.user.js
```

## 使い方

Tampermonkeyでスクリプトを有効にした状態で、EDHRECの統率者ページやカードページを開きます。

カード画像が差し替わると、カード画像の下に小さな操作バーが出ます。

- カード名: Scryfallを開く
- コピー: カード名をクリップボードへコピーする
- ★: お気に入りに追加、または解除する
- 晴 / BM / SS / 東 / メ: 各ショップやメルカリで検索する

右下の `★ お気に入り` ボタンを押すと、お気に入りパネルが開きます。

パネル内の `全部コピー` を押すと、お気に入りに入れたカード名を改行区切りでまとめてコピーできます。ExcelやGoogleスプレッドシートに貼ると、1行ずつカード名が入ります。

## 表示について

カード画像の上に文字を重ねると、カード名やボタンが読みにくくなることがありました。

そのため、現在のバージョンでは操作バーをカード画像の下に表示しています。カード画像を隠さず、日本語名やコピー用ボタンを読めるようにするためです。

EDHREC本来の価格やsynergy表示がカード画像の下にある場合は、その表示を先に残し、さらに下へ操作バーを追加します。

横長に表示されるサムネイルのように、カード形ではない枠に差し替え画像を入れるとカード内の文字が潰れて読みにくくなります。そのため、そういう画像は差し替え対象外にしています。

また、EDHRECのページ上部にある統率者画像のように、カードリンクではなく画像だけで表示されているものもあります。この場合はScryfall画像URLからカードIDを読み取り、画像だけを差し替えます。ページ上部のレイアウトを壊しやすいため、そこには操作バーを出していません。

## 日本語版がないカードについて

Scryfallに日本語印刷版が見つからないカードは、Scryfallの英語通常版へ差し替えます。

日本語化できないカードでも、特殊アートではない通常版に寄せて表示し、Scryfallやショップ検索への導線は使えるようにしています。

## Scryfall APIへの配慮

このスクリプトはブラウザからScryfall APIへ問い合わせます。

Scryfallの公式FAQでは、`api.scryfall.com` へのアクセスを10リクエスト/秒未満に抑えること、適切なヘッダーを使うこと、`429 Too Many Requests` を無視しないことが案内されています。

このスクリプトでは、次のようにしています。

- API呼び出しは直列キューで実行する
- リクエスト間隔は `110ms` 以上空ける
- 最大でも約 `9.09 req/sec` に抑える
- 同じカード名への同時リクエストは1つにまとめる
- 検索結果はブラウザの `localStorage` に7日間キャッシュする
- 1回のスキャンで先読みするカード数は最大80件にする
- 画像の先読みは1回のスキャンで最大40件にする
- `429 Too Many Requests` では `Retry-After` を見て待つ
- 一時的な通信失敗や5xx系エラーは、少し時間をおいて最大2回まで再試行する

ブラウザ上で動くユーザースクリプトなので、`User-Agent` はブラウザ標準のものを使います。サーバーやNode.jsで同じ仕組みを作る場合は、アプリ名や連絡先が分かる `User-Agent` を別途設定してください。

## メルカリリンクについて

メルカリへのリンクは通常の検索リンクにしています。

メルカリアンバサダーなどのアフィリエイトリンクを使う場合は、公式の規約とガイドラインを確認し、自分が管理するブログやSNSなどの媒体で、必要なPR表記を添えて利用してください。

ユーザースクリプト内に作者のアフィリエイトリンクを固定で入れる形にはしていません。

## 保存されるデータ

ブラウザの `localStorage` に次のデータを保存します。

- Scryfall検索結果キャッシュ: `edhrec-ja-image-cache-v2`
- お気に入りカード一覧: `edhrec-ja-image-favorites-v1`

お気に入り情報を外部サーバーへ送信する機能はありません。

## うまく動かないとき

まず、次を確認してください。

- Tampermonkeyでスクリプトが有効になっている
- 対象URLが `https://edhrec.com/*` または `https://www.edhrec.com/*` である
- ページを再読み込みしてみる
- ブラウザ拡張や広告ブロッカーが `api.scryfall.com` への通信を止めていない
- Greasy ForkまたはGitHubの最新版を使っている

EDHREC側のHTML構造が大きく変わると、一部のカードでうまく差し替わらない可能性があります。その場合はGitHubのIssueで対象ページURLを共有してください。

```text
https://github.com/soichirow/edhrec-ja-images/issues
```

## 実装のメモ

スクリプトはEDHREC上の画像やカードリンクから英語カード名を拾い、Scryfall APIで日本語印刷版を探します。

主に見ている情報は次の通りです。

- 画像の `alt`
- 画像の `title`
- カードリンクの `aria-label`
- カードリンクの `title`
- EDHRECのカードURL
- Scryfall画像URL内のカードID

日本語印刷版を探すときは、できるだけ通常版に近い画像を選ぶようにしています。ショーケース、ボーダーレス、拡張アート、プロモ系などは避けています。

テストでは、Node.jsの構文チェックと仕様チェックに加えて、実ブラウザで表示確認するE2Eテストも入れています。

E2Eテストでは、表示確認用ページをブラウザで開き、次のことを確認しています。

- 通常カードの画像が差し替わる
- 画像下に操作バーが出る
- ショップリンクが表示される
- ページ上部の統率者風画像は画像だけ差し替わる
- 操作バーがカード画像に重ならない
- お気に入り登録が動く

## まとめ

EDHRECは統率者デッキを考えるときにとても便利ですが、日本語名でカードを管理したい人にとっては、英語名から日本語名へ変換する作業が少し手間です。

このスクリプトを入れると、EDHRECを見ながらカード画像、日本語名コピー、お気に入り管理、ショップ検索までまとめてできます。

デッキを組むときの小さな手間が減るので、同じように日本語名でカード管理している人は試してみてください。
