# EDHRECで日本語カード画像を見ながら統率者デッキを組めるようにするTampermonkeyスクリプト

EDHRECは、Magic: The Gatheringの統率者デッキを組むときに、統率者ごとの採用カードや相性のよいカードを調べられるサイトです。

カード候補を探すのにとても便利ですが、海外サイトなのでカード画像やカード名は基本的に英語で表示されます。

そのまま使っていると、「このカード、日本語名なんだっけ？」を何度も調べることになります。採用候補をメモしたり、あとで買うカードを探したりするとき、そのひと手間が少しずつ積み重なります。

このスクリプトは、EDHRECのカード画像をできるだけ日本語版に差し替えて、日本語名のコピーやショップ検索をその場でしやすくするためのものです。

EDHRECの使い方を大きく変えるのではなく、日本語名でデッキを管理したい人の手元作業を少し軽くすることを目指しています。

ブラウザにTampermonkeyを入れて、Greasy Forkからスクリプトをインストールすると使えます。Tampermonkeyは、ブラウザ上でユーザースクリプトを動かすための拡張機能です。

## こんな人に向いています

- EDHRECを見ながら、日本語名でデッキリストや買い物リストを作りたい
- 英語名から日本語名を毎回調べ直す時間を減らしたい
- カード画像は見やすいまま、コピーや検索だけ手早くしたい
- 気になったカードをあとでまとめて確認したい

## 何が楽になるか

- EDHREC上のカード画像が、可能な範囲で日本語版に置き換わります
- 日本語版がないカードも、Scryfallの通常版画像で見やすく表示します
- カード名のコピー、お気に入り管理、ショップ検索をEDHREC上でまとめて使えます
- 操作バーはカード画像の直後に置き、EDHREC本来の表示をなるべく崩さないようにします
- 一度調べたカード情報をブラウザ内に保存して、同じカードを見たときの表示を少し速くします

## インストール方法

まずはGreasy Forkから入れるのがおすすめです。

1. Tampermonkeyを入れる  
   https://www.tampermonkey.net/
2. スクリプトページを開く  
   https://greasyfork.org/ja/scripts/580860-edhrec-japanese-card-image-replacer
3. `Install this script` を押す
4. Tampermonkeyの確認画面で `インストール` を押す
5. EDHRECを開く  
   https://edhrec.com/

これでEDHREC上のカード画像が順番に差し替わります。

## 使い方

Tampermonkeyでスクリプトを有効にした状態で、EDHRECの統率者ページやカードページを開きます。

カード画像が差し替わると、カード画像の下に小さな操作バーが出ます。デッキ候補を眺めながら、必要なカードだけコピーしたり、あとで見返すためにお気に入りへ入れたりできます。

- Scryfall: 個別ページを開く
- コピー: カード名をクリップボードへコピーする
- ★: お気に入りに追加、または解除する
- 晴 / BM / SS / 東 / メ: 晴れる屋、BIG MAGIC、シングルスター、東京MTG、メルカリで検索する

右下の `★ お気に入り` ボタンを押すと、お気に入りパネルが開きます。

パネル内の `全部コピー` を押すと、お気に入りに入れたカード名を改行区切りでまとめてコピーできます。ExcelやGoogleスプレッドシートに貼れば、そのまま買い物リストや調整用メモとして使えます。

## 表示で気をつけていること

カード画像の上に文字やボタンを重ねると、肝心のカードが読みにくくなります。

そのため、操作バーはカード画像の直後に表示しています。日本語名コピーや検索ボタンを使えるようにしつつ、カード画像そのものはできるだけ隠さないようにしています。

基本の並びは、カード画像、操作バー、EDHREC本来のカード名や価格、synergy表示です。EDHREC本体のお気に入り表示と、追加したお気に入りボタンが近くなりすぎないようにしています。

横長に表示されるサムネイルのように、カード形ではない枠に差し替え画像を入れるとカード内の文字が潰れて読みにくくなります。そのため、そういう画像は差し替え対象外にしています。

また、EDHRECのページ上部にある統率者画像のように、カードリンクではなく画像だけで表示されているものもあります。この場合は画像だけを差し替え、ページ上部の見た目を保つために操作バーは出していません。

## 日本語版がないカード

Scryfallに日本語印刷版が見つからないカードは、Scryfallの英語通常版へ差し替えます。

日本語版がないカードを無理に日本語化することはできませんが、できるだけ通常版に近い画像で表示し、Scryfallやショップ検索への導線は使えるようにしています。

## 外部サービスへの配慮

このスクリプトはカード情報をScryfallから取得しています。

Scryfallの公式FAQでは、`api.scryfall.com` へのアクセスを10リクエスト/秒未満に抑えること、アクセスが多すぎると判断されたときに待つこと、`429 Too Many Requests` を無視しないことが案内されています。

このスクリプトでは、次のようにしています。

- カード情報は一度にまとめて取りに行かず、少しずつ問い合わせる
- 同じカードを何度も探し直さないようにする
- 検索結果はしばらくブラウザ内に保存して、同じページを見たときの通信を減らす
- アクセスが多すぎると判断された場合は、時間をおいてから再試行する

大量のカード情報を集めるためのツールではなく、EDHRECを見ているページ上のカードを少しずつ補助するためのスクリプトです。

## 保存されるデータ

次のデータは、使っているブラウザ内に保存されます。

- Scryfall検索結果の一時データ
- お気に入りカード一覧

お気に入り情報を外部サーバーへ送信する機能はありません。別のブラウザや別の端末には自動では引き継がれないので、必要なカード名は `全部コピー` でメモしておくと安心です。

## 困ったとき

画像がすぐに変わらないときや、操作バーが出ないときは、まず次を確認してみてください。

- Tampermonkeyでスクリプトが有効になっている
- 対象URLが `https://edhrec.com/*`、`https://www.edhrec.com/*`、`https://tagger.scryfall.com/*` のいずれかである
- ページを再読み込みしてみる
- ブラウザ拡張や広告ブロッカーが `api.scryfall.com` への通信を止めていない
- Greasy Forkの最新版を使っている

EDHREC側の表示が大きく変わると、一部のカードでうまく差し替わらない可能性があります。再読み込みしても直らない場合は、GitHubのIssueで対象ページURLを共有してもらえると助かります。

```text
https://github.com/soichirow/edhrec-ja-images/issues
```

## まとめ

EDHRECは便利ですが、日本語名でカードを管理したい人にとっては、英語名から日本語名へ変換する作業が小さな負担になります。

このスクリプトを入れると、EDHRECを見ながら日本語カード画像を確認し、カード名のコピー、お気に入り管理、ショップ検索までその場でできます。

「EDHRECは使いたいけど、日本語名で管理するのが少し面倒」と感じている人には、かなり相性がいいと思います。
